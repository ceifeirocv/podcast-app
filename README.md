# Podcast App

Short overview of the project.

## Environment

This project uses a few environment variables for third-party integrations. For Podcast Index, the app expects two variables during development:

- EXPO_PUBLIC_PODCAST_INDEX_KEY
- EXPO_PUBLIC_PODCAST_INDEX_SECRET

These are public Expo variables (EXPO_PUBLIC_*) and will be included in the client bundle. For more details, local setup instructions, and security recommendations, see: doc/research/podcast-index-env.md
